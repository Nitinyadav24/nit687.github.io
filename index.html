Guide to Prompt Writing

Prompt Writing Best Practices

  • Separate instruction and context: Clearly distinguish between the instruction and the context with markers such as ### or """.
    Example:
    Less effective ❌: Translate the text below into French. {text input here}
    Better ✅: Translate the text below into French. Text: """ {text input here} """
  • Be specific and detailed: Give precise instructions about the desired context, outcome, length, format, style, etc.
    Example:
    Less effective ❌: Write a story about a young wizard.
    Better ✅: Write a suspenseful short story about a young wizard who has just discovered his powers, set in a modern urban environment.
  • Articulate the desired output format through examples: Show the desired format in examples.
    Example:
    Less effective ❌: Summarize the key points from the text below.
    Better ✅: Summarize the key points from the text below. Desired format: Key Point 1: _, Key Point 2: _, etc. Text: {text}
  • Zero-shot, then few-shot, then fine-tune: Start with zero-shot prompts, then few-shot examples, and then fine-tuning if needed.
  • Reduce “fluffy” and imprecise descriptions: Be concise and clear in your instructions.
    Example:
    Less effective ❌: Write a decent amount of content about the topic.
    Better ✅: Write a detailed 500-word article about the topic.
  • Say what to do instead of just what not to do: Clearly instruct on what needs to be done instead of just highlighting what should be avoided.
    Example:
    Less effective ❌: The character should not be sad or upset.
    Better ✅: The character should exhibit feelings of excitement and joy.
  • For code generation, use “leading words”: Use certain leading words to steer the model towards a particular pattern.
    Example:
    Less effective ❌: Write a function to perform the following task.
    Better ✅: def function_name(): # Write a function to perform the following task.
